# 🛠️ Projeto 02: Calculadora Manual de Binários

Domine a conversão de números decimais para o "idioma" básico do processador através de um exercício de sistematização.

## 📝 Contexto
Saber converter números para binário é como saber conjugar verbos em um novo idioma. É a base para entender como chips de memória, endereços IP e cores RGB funcionam internamente.

## 🎯 Objetivo
Realizar a conversão completa e documentada de um número decimal de significado pessoal (sua idade, o dia do seu nascimento ou os dois últimos dígitos do seu ano de nascimento).

## 📋 Requisitos
1. **Escolha do Número**: Escolha um número decimal entre **10 e 100**.
2. **Método das Divisões**: Realize o cálculo passo a passo usando divisões sucessivas por 2.
3. **Documentação**:
    - Mostre todas as divisões.
    - Destaque os **restos** obtidos.
    - Identifique o bit **MSB** (último resto) e o bit **LSB** (primeiro resto).
4. **Verificação**: Tente somar os pesos ($2^n$) do binário resultante para confirmar se ele volta ao seu número decimal original.

## 🚚 Entrega
Crie um documento (pode ser manuscrito com foto ou digital) contendo:
- O número escolhido e por que o escolheu.
- O quadro de divisões completo.
- O número resultante em binário.
- A prova real do cálculo (soma dos pesos).
