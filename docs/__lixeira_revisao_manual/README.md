# 🗑️ Lixeira de Revisão Manual

Esta pasta contém todos os arquivos e pastas do site antigo que foram movidos durante a reestruturação do projeto. 

**NENHUM ARQUIVO DAQUI DEVE SER EXCLUÍDO AUTOMATICAMENTE.**

## Objetivo
Permitir a auditoria humana e a recuperação de conteúdo caso necessário.

## Conteúdo
- `aulas/`: Conteúdo antigo das aulas.
- `slides/`: Slides antigos.
- `assets/`: Imagens e arquivos estáticos antigos.
- `README.md`: Readme original do projeto.
- `_config.yml`: Configuração antiga do Jekyll.
- `curso.md`, `ementa.md`, `topicos/`: Arquivos de planejamento e conteúdo antigo.

## Instruções
1. Verifique se o conteúdo necessário foi migrado para a nova estrutura em `../`.
2. Após a confirmação de que tudo está correto e nada mais é necessário, esta pasta pode ser excluída manualmente.
