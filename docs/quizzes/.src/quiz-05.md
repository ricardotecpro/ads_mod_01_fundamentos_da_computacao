# 🔢 Quiz 05: Sistema Hexadecimal (Base 16)

Avalie seu domínio sobre a base numérica mais utilizada por desenvolvedores e designers: o Hexadecimal.

1. Qual a base numérica do sistema hexadecimal?
    - [ ] 2
    - [ ] 10
    - [x] 16
    - [ ] 32
    *feedback: Hexa (6) + Deci (10) = 16 símbolos.*

2. Quais símbolos o sistema hexadecimal utiliza para representar os valores de 10 a 15?
    - [ ] 10, 11, 12, 13, 14, 15
    - [ ] X, Y, Z, W, K, L
    - [x] A, B, C, D, E, F
    - [ ] !, @, #, $, %, &
    *feedback: Utilizamos as primeiras 6 letras do alfabeto para evitar o uso de dois dígitos decimais.*

3. Qual o valor decimal correspondente à letra **A** em hexadecimal?
    - [ ] 1
    - [x] 10
    - [ ] 11
    - [ ] 16
    *feedback: A é o primeiro símbolo após o dígito 9.*

4. Qual o valor decimal correspondente à letra **F** em hexadecimal?
    - [ ] 10
    - [ ] 14
    - [x] 15
    - [ ] 16
    *feedback: F é o último símbolo da base 16, valendo 15 ($16-1$).*

5. Qual a relação matemática entre a Base 2 e a Base 16?
    - [ ] $2 \times 8 = 16$
    - [x] $2^4 = 16$
    - [ ] $16 = 2^2$
    - [ ] $2+14 = 16$
    *feedback: Como $2^4 = 16$, cada dígito hexadecimal resume exatamente 4 bits.*

6. O termo "Nibble" refere-se a um conjunto de quantos bits?
    - [ ] 2 bits
    - [x] 4 bits
    - [ ] 8 bits
    - [ ] 16 bits
    *feedback: Um Nibble é metade de um Byte e corresponde a exatamente um dígito Hexa.*

7. Em cores Web (CSS), o que representa o código `#FF0000`?
    - [ ] Verde Máximo
    - [ ] Azul Máximo
    - [x] Vermelho Máximo
    - [ ] Branco
    *feedback: Os pares representam R-G-B. FF no primeiro par é Vermelho (Red) no máximo.*

8. Qual o valor decimal resultante da conversão do hexadecimal `10`?
    - [ ] 10
    - [ ] 12
    - [x] 16
    - [ ] 20
    *feedback: $1 \times 16^1 + 0 \times 16^0 = 16 + 0 = 16$.*

9. Por que o Hexadecimal é preferido em relação ao Octal em sistemas modernos?
    - [ ] É mais bonito
    - [ ] É mais antigo
    - [x] Um Byte (8 bits) pode ser representado exatamente por 2 dígitos Hexa
    - [ ] Usa menos letras
    *feedback: O alinhamento com a arquitetura de 8 bits (Byte) torna o Hexa extremamente prático.*

10. Qual o valor hexadecimal do número decimal 255?
    - [ ] F0
    - [ ] EF
    - [ ] FA
    - [x] FF
    *feedback: $255 = 15 \times 16 + 15$, o que corresponde a FF em hexadecimal.*
