# 🖥️ Quiz 14: Sistemas Operacionais

Teste seu conhecimento sobre o software mais importante do computador: aquele que gerencia todos os outros.

1. Qual a função básica de um Sistema Operacional (SO)?
    - [ ] Criar desenhos e planilhas
    - [ ] Aumentar a velocidade da internet
    - [x] Atuar como intermediário entre o hardware e o usuário/aplicativos
    - [ ] Limpar o pó de dentro do gabinete
    *feedback: O SO é o gerente que permite que softwares usem o hardware de forma organizada.*

2. Como é chamada a parte central e mais crítica de um Sistema Operacional, que fica sempre na memória RAM?
    - [ ] Shell
    - [x] Kernel
    - [ ] Desktop
    - [ ] Bio
    *feedback: O Kernel (núcleo) gerencia a CPU, a memória e os dispositivos básicos.*

3. O que é "Gerenciamento de Processos" em um SO?
    - [ ] Limpar o disco rígido de arquivos velhos
    - [ ] Instalar novos programas
    - [x] Decidir qual programa pode usar o processador (CPU) em cada momento
    - [ ] Trocar a cor das janelas
    *feedback: Isso permite que o computador pareça fazer várias coisas ao mesmo tempo (multitarefa).*

4. O que acontece se dois programas tentarem usar o mesmo espaço exato na memória RAM ao mesmo tempo?
    - [ ] O computador fica mais rápido
    - [x] Ocorre um conflito que pode travar o sistema ou fechar os programas
    - [ ] Os dados se fundem em um novo arquivo
    - [ ] A tela fica preta para economizar energia
    *feedback: O SO deve isolar a memória de cada programa por segurança e estabilidade.*

5. Qual o papel fundamental dos "Drivers" (Controladores de Dispositivos)?
    - [ ] Servem para navegar na internet
    - [ ] São programas de edição de vídeo
    - [x] Explicar ao SO como se comunicar com um hardware específico (ex: impressora, placa de vídeo)
    - [ ] São vírus que lentificam o PC
    *feedback: Sem o driver correto, o SO não sabe quais comandos enviar para o hardware funcionar.*

6. A sigla GUI (Graphical User Interface) refere-se a:
    - [ ] Usar apenas o teclado para comandos
    - [x] Interfaces com janelas, ícones, mouse e elementos visuais
    - [ ] O código fonte do Kernel
    - [ ] Um tipo de cabo de rede
    *feedback: O Windows, macOS e o Android são exemplos focados em GUI.*

7. O que caracteriza uma interface CLI (Command Line Interface)?
    - [ ] Uso intenso de cores e animações
    - [ ] Necessidade obrigatória de um mouse gamer
    - [x] Interação baseada inteiramente em comandos de texto digitados no teclado
    - [ ] Só funciona se houver internet
    *feedback: Terminais e Prompts de comando são exemplos de interfaces CLI.*

8. Qual o nome da camada de software que recebe os comandos do usuário (via texto ou cliques) e os envia para o Kernel?
    - [ ] Hardware
    - [ ] Firmware
    - [x] Shell
    - [ ] Driver
    *feedback: O Shell é a "casca" que envolve o Kernel para interagir com o mundo externo.*

9. FAT32, NTFS e EXT4 são exemplos de:
    - [ ] Marcas de memória RAM
    - [ ] Tipos de monitores
    - [x] Sistemas de Arquivos (organização de dados no disco)
    - [ ] Versões do Linux
    *feedback: Eles definem como os nomes de arquivos e pastas são guardados fisicamente.*

10. Se um programa para de responder ("trava"), qual componente do SO é responsável por permitir que o usuário force o seu fechamento?
    - [ ] O Driver da tela
    - [x] O Gerenciador de Processos (Escalonador)
    - [ ] O Mouse
    - [ ] A BIOS
    *feedback: O SO mantém o controle e pode interromper um processo problemático.*
