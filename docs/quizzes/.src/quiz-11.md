# 🔌 Quiz 11: Circuitos Lógicos

Teste sua capacidade de identificar componentes lógicos e entender a engenharia por trás do hardware do computador.

1. Qual símbolo visual representa a porta **NOT** nos diagramas lógicos?
    - [ ] Um retângulo com "D" dentro
    - [x] Um triângulo com uma pequena bolinha na ponta
    - [ ] Uma seta pontiaguda
    - [ ] Um círculo com um X
    *feedback: A "bolinha" na extremidade simboliza a inversão do sinal elétrico.*

2. A porta lógica que possui o formato de um "D" alongado é a:
    - [ ] OR
    - [x] AND
    - [ ] NOT
    - [ ] XOR
    *feedback: O símbolo do AND é padronizado por um formato em D nas normas ANSI.*

3. Qual componente é responsável por realizar a soma de dois bits individuais e gerar um "vai-um"?
    - [ ] Subtrator
    - [ ] Decodificador
    - [x] Half-Adder (Meio Somador)
    - [ ] Multiplexador
    *feedback: O Meio Somador é a unidade básica de cálculo aritmético da CPU.*

4. Em um Half-Adder (Meio Somador), qual porta lógica é utilizada para calcular o bit da **SOMA**?
    - [ ] AND
    - [ ] OR
    - [x] XOR
    - [ ] NAND
    *feedback: O XOR resulta em 1 apenas se houver apenas um bit '1' (estética da soma sem carry).*

5. Qual porta lógica do Half-Adder gera o bit de **CARRY** (Vai-um)?
    - [ ] XOR
    - [ ] OR
    - [x] AND
    - [ ] NOT
    *feedback: O carry só ocorre se AMBOS os bits forem 1, exatamente como o comportamento de uma porta AND.*

6. Qual a principal diferença entre um Half-Adder e um Full-Adder (Somador Completo)?
    - [ ] O Full-Adder é mais rápido
    - [ ] O Half-Adder usa 10 bits
    - [x] O Full-Adder aceita um "Carry In" (Vem-um) da coluna anterior
    - [ ] O Half-Adder só funciona em Linux
    *feedback: Para somar números longos, precisamos levar em conta o vai-um que veio da casa da direita.*

7. Milhares de portas lógicas compactadas em uma única pastilha de silício formam um:
    - [ ] Capacitor
    - [ ] Transistor individual
    - [x] Circuito Integrado (Chip)
    - [ ] Resistor
    *feedback: CIs ou Chips permitem a miniaturização extrema dos computadores modernos.*

8. Qual o nome do componente eletrônico básico que atua como um "interruptor" para os bits e forma as portas lógicas?
    - [ ] Válvula de vácuo
    - [x] Transistor
    - [ ] Sensor de luz
    - [ ] Cabo de fibra ótica
    *feedback: O transistor é a unidade fundamental do processamento digital moderno.*

9. O material químico mais utilizado na fabricação de chips por ser um semicondutor é o:
    - [ ] Ouro
    - [ ] Ferro
    - [x] Silício
    - [ ] Alumínio
    *feedback: O silício (derivado da areia) tem propriedades que permitem controlar o fluxo de eletricidade.*

10. Se ligarmos a saída de uma porta AND na entrada de uma porta NOT, que porta lógica combinada teremos?
    - [ ] NOR
    - [x] NAND
    - [ ] XOR
    - [ ] OR
    *feedback: O NOT do AND (Não-E) resulta no operador NAND.*
