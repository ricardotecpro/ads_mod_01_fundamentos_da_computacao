# 🚀 Quiz 16: Introdução à Programação

Teste seu conhecimento sobre como traduzimos ideias humanas para a linguagem das máquinas através da programação.

1. O que é "Programar" em informática?
    - [ ] Consertar peças físicas do computador
    - [x] Dar uma sequência de instruções para o computador executar tarefas
    - [ ] Navegar apenas por redes sociais
    - [ ] Criar desenhos e imagens manuais
    *feedback: Programar é o ato de escrever software para resolver problemas ou automatizar processos.*

2. Como chamamos o tradutor de código que transforma todo o seu programa de uma vez em um arquivo executável (.exe)?
    - [ ] Shell
    - [ ] Navegador
    - [x] Compilador
    - [ ] Driver
    *feedback: Linguagens como C, C++ e Go usam compiladores para gerar arquivos prontos para rodar.*

3. Qual a principal característica de uma linguagem **Interpretada** (como Python ou JavaScript)?
    - [ ] Ela não precisa de hardware para rodar
    - [ ] Ela é mais rápida que a compilada
    - [x] O código é traduzido e executado linha por linha em tempo real
    - [ ] Ela só aceita números binários
    *feedback: O interpretador lê o script e executa na hora, o que facilita o desenvolvimento e testes.*

4. Na programação, o que é uma **Variável**?
    - [ ] Uma parte do computador que se move
    - [x] Um espaço reservado na memória para armazenar e manipular dados
    - [ ] Um erro que altera o resultado aleatoriamente
    - [ ] Um tipo de monitor que muda de cor
    *feedback: Pense nas variáveis como janelas ou nomes que guardam valores temporários.*

5. Para que servem os **Laços de Repetição** (Loops)?
    - [ ] Para desligar o computador mais rápido
    - [x] Para executar o mesmo bloco de código várias vezes seguidas
    - [ ] Para mudar a cor das janelas do sistema
    - [ ] Para apagar arquivos antigos
    *feedback: Loops economizam trabalho, repetindo tarefas automáticas de forma eficiente.*

6. O que representa o termo "Bug" no desenvolvimento de software?
    - [ ] Um inseto real dentro da CPU
    - [x] Um erro de lógica ou sintaxe no código que causa comportamento inesperado
    - [ ] Um vírus altamente perigoso
    - [ ] Uma ferramenta de depuração
    *feedback: O termo surgiu historicamente quando um inseto real causou um curto em um computador antigo.*

7. As linguagens de **Alto Nível** (como Python ou Java) são caracterizadas por:
    - [ ] Serem escritas apenas em binário
    - [x] Serem mais próximas da linguagem humana e mais fáceis de ler
    - [ ] Funcionarem apenas em supercomputadores
    - [ ] Serem invisíveis para o usuário
    *feedback: Elas abstraem a complexidade do hardware para facilitar a vida do programador.*

8. Qual o objetivo tradicional de se escrever um programa "Hello World" (Olá Mundo)?
    - [ ] Ganhar dinheiro rápido
    - [x] Testar se o ambiente de programação e a linguagem estão funcionando corretamente
    - [ ] Criar uma inteligência artificial completa
    - [ ] Hackear o sistema operacional
    *feedback: É o teste mais simples possível para confirmar que "está tudo pronto" para começar.*

9. Qual desses componentes NÃO faz parte da anatomia básica da lógica de um programa comum?
    - [ ] Variáveis
    - [ ] Estruturas Condicionais (IF)
    - [ ] Funções
    - [x] Teclado mecânico
    *feedback: O teclado é o hardware de entrada, não o componente lógico do código.*

10. Qual a fase final do desenvolvimento de um software, antes de entregá-lo ao usuário?
    - [ ] Compilação
    - [ ] Escrita do código
    - [x] Testes e Depuração (Debugging)
    - [ ] Formatação do HD
    *feedback: Testar garante que o programa atenda aos requisitos e esteja livre de erros críticos.*
