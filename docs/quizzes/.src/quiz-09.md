# 🧠 Quiz 09: Lógica Booleana

Teste sua capacidade de pensar como um processador e tomar decisões usando os operadores lógicos fundamentais.

1. Quem foi o matemático que criou o sistema de lógica que hoje leva seu nome e é a base da computação?
    - [ ] Alan Turing
    - [ ] John von Neumann
    - [x] George Boole
    - [ ] Blaise Pascal
    *feedback: George Boole publicou em 1854 as leis do pensamento que hoje regem a lógica digital.*

2. No contexto da lógica booleana, o valor binário `0` é equivalente a:
    - [ ] Verdadeiro
    - [x] Falso
    - [ ] Positivo
    - [ ] Indefinido
    *feedback: O 0 representa a ausência de sinal, ou o estado logico Falso (False).*

3. Qual a função principal do operador **NOT** (NÃO)?
    - [ ] Somar dois bits
    - [ ] Multiplicar o valor por zero
    - [x] Inverter o estado lógico da entrada
    - [ ] Comparar se dois valores são iguais
    *feedback: Se entra Verdadeiro, sai Falso. Se entra Falso, sai Verdadeiro.*

4. Qual operador lógico só resulta em **Verdadeiro** se **TODAS** as suas entradas forem verdadeiras?
    - [ ] OR (OU)
    - [x] AND (E)
    - [ ] XOR (OU Exclusivo)
    - [ ] NOT (NÃO)
    *feedback: O AND funciona como uma condição acumulativa e obrigatória.*

5. O operador **OR** (OU) resulta em Verdadeiro quando:
    - [ ] Todas as entradas são falsas
    - [ ] Apenas a primeira entrada é verdadeira
    - [x] Pelo menos uma das entradas é verdadeira
    - [ ] As entradas são diferentes
    *feedback: O OR é inclusivo; basta uma condição ser aceita para o resultado ser positivo.*

6. Qual o resultado da expressão lógica `1 AND 0`?
    - [ ] 1
    - [x] 0
    - [ ] 2
    - [ ] Verdadeiro
    *feedback: Na lógica AND, o zero "anula" o resultado, tornando-o falso.*

7. Qual o resultado da expressão lógica `1 OR 0`?
    - [ ] 0
    - [x] 1
    - [ ] Falso
    - [ ] 10
    *feedback: Como o OR exige apenas uma entrada verdadeira, o 1 garante o resultado 1.*

8. Na ausência de parênteses, qual a ordem correta de prioridade (hierarquia) dos operadores?
    - [ ] AND > OR > NOT
    - [ ] OR > AND > NOT
    - [x] NOT > AND > OR
    - [ ] A ordem não importa em lógica
    *feedback: Resolvemos primeiro a inversão, depois a conjunção e por fim a disjunção.*

9. Em uma analogia elétrica, qual operador pode ser representado por dois interruptores ligados em **SÉRIE** (um após o outro)?
    - [ ] OR
    - [x] AND
    - [ ] NOT
    - [ ] XOR
    *feedback: Para a lâmpada acender, o interruptor A **E** o interruptor B precisam estar fechados.*

10. Qual o resultado final da expressão `NOT (NOT 1)`?
    - [ ] 0
    - [x] 1
    - [ ] Falso
    - [ ] Indefinido
    *feedback: A dupla negação anula a si mesma, retornando ao valor original.*
