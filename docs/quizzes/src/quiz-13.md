# 💾 Quiz 13: Memória e Armazenamento

Avalie seu entendimento sobre como os dados são guardados e acessados nos computadores modernos.

1. Qual a principal característica que diferencia a Memória RAM de um HD ou SSD?
    - [ ] A RAM é mais lenta
    - [ ] A RAM guarda dados permanentemente
    - [x] A RAM é volátil (perde dados sem energia)
    - [ ] A RAM é feita de metal magnético
    *feedback: A volatilidade é a característica central da RAM; ela é rápida, mas temporária.*

2. Na hierarquia de memória, qual componente é geralmente o MAIS RÁPIDO e possui o MAIOR CUSTO por byte?
    - [ ] SSD
    - [ ] RAM
    - [x] Registradores e Cache
    - [ ] Pendrive
    *feedback: Quanto mais perto do processamento (CPU), mais rápida e cara é a memória.*

3. Onde um arquivo que você acabou de "Salvar" no Word fica armazenado permanentemente?
    - [ ] Na CPU
    - [ ] Na Memória RAM
    - [x] Na Memória Secundária (HD/SSD)
    - [ ] Na Placa de Vídeo
    *feedback: Memórias secundárias são não-voláteis e mantêm os dados após o desligamento.*

4. Qual a principal vantagem de um SSD em relação a um HD tradicional?
    - [ ] É mais barato por GigaByte
    - [ ] Tem maior durabilidade magnética
    - [x] Não possui partes móveis e é muito mais rápido no acesso aos dados
    - [ ] Funciona apenas em notebooks
    *feedback: SSDs usam chips eletrônicos, eliminando o atraso mecânico de discos giratórios.*

5. A tecnologia de discos rígidos (HD) tradicionais baseia-se em:
    - [ ] Feixes de laser
    - [x] Magnetismo em discos metálicos giratórios
    - [ ] Circuitos de silício fixos
    - [ ] Baterias de lítio
    *feedback: O HD usa cabeças de leitura físicas sobre discos magnéticos, como um toca-discos moderno.*

6. Qual a função da Memória Cache (L1, L2, L3)?
    - [ ] Guardar as fotos do usuário
    - [x] Reduzir o gargalo de velocidade entre a CPU ultra-rápida e a RAM mais lenta
    - [ ] Aumentar a resolução da tela
    - [ ] Fazer o backup do Windows
    *feedback: A Cache guarda cópias de dados frequentes para que a CPU não precise esperar pela RAM.*

7. O que é a Memória Virtual?
    - [ ] Um chip especial comprado em lojas
    - [ ] Uma memória que só existe na internet (nuvem)
    - [x] Uma técnica do SO que usa parte do HD/SSD como se fosse RAM quando esta acaba
    - [ ] Um simulador de memórias antigas
    *feedback: Ela permite rodar mais programas do que a RAM física suportaria, sacrificando velocidade.*

8. No sistema binário usado em informática (padrão KiB), 1 Kilobyte (Kibibyte) corresponde a:
    - [ ] 1.000 Bytes
    - [x] 1.024 Bytes
    - [ ] 1.000.000 Bytes
    - [ ] 512 Bytes
    *feedback: Computadores usam potências de 2, e $2^{10} = 1024$.*

9. Quantos MegaBytes (MB) aproximadamente existem em 1 GigaByte (GB)?
    - [ ] 100
    - [ ] 500
    - [x] 1.000 (ou 1.024)
    - [ ] 10.000
    *feedback: A escala sobe de 1.000 em 1.000 (ou 1.024 em 1.024).*

10. Por que um SSD de "500 GB" anunciado pelo fabricante muitas vezes aparece como "465 GB" no Windows?
    - [ ] O Windows rouba espaço para o sistema
    - [ ] O SSD veio com defeito de fábrica
    - [x] Fabricantes usam base 10 (1000) e sistemas usam base 2 (1024)
    - [ ] Metade do espaço é invisível por segurança
    *feedback: Essa diferença de cálculo entre decimal e binário causa a percepção de "perda" de espaço.*
