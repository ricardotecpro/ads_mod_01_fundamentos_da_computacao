# 🏛️ Quiz 12: Arquitetura de Computadores

Avalie seu conhecimento sobre o "esqueleto" e o "motor" que fazem qualquer computador funcionar.

1. Qual o nome da arquitetura fundamental proposta em 1945 que define a organização da maioria dos computadores modernos?
    - [ ] Arquitetura Harvard
    - [x] Arquitetura de Von Neumann
    - [ ] Arquitetura RISC
    - [ ] Arquitetura ARM
    *feedback: John von Neumann descreveu o modelo onde processamento e memória são separados, mas compartilham o mesmo espaço para dados e instruções.*

2. A principal característica do Modelo de Von Neumann é:
    - [ ] Não usar eletricidade
    - [ ] Ter memória infinita
    - [x] Armazenar tanto os dados quanto as instruções do programa na mesma memória RAM
    - [ ] Usar apenas transistores de silício
    *feedback: Isso permitiu a criação de computadores "programáveis" que mudam de tarefa sem troca de hardware.*

3. A sigla CPU (Central Processing Unit) pode ser traduzida para o português como:
    - [ ] Unidade Central de Portas
    - [x] Unidade Central de Processamento
    - [ ] Cabo de Processamento Único
    - [ ] Centro de Programação Universal
    *feedback: Ela é o cérebro do computador, onde todos os cálculos e decisões são feitos.*

4. Qual componente da CPU é responsável por realizar cálculos matemáticos e comparações lógicas?
    - [ ] Unidade de Controle (UC)
    - [ ] Registradores
    - [x] Unidade Lógica e Aritmética (ALU)
    - [ ] Memória Cache
    *feedback: A ALU (Arithmetic Logic Unit) é o "músculo" matemático do processador.*

5. Qual a função da Unidade de Controle (UC) dentro da CPU?
    - [ ] Guardar os arquivos do usuário
    - [x] Buscar instruções na memória, decodificá-las e coordenar os outros componentes
    - [ ] Resfriar o processador através de ventoinhas
    - [ ] Multiplicar o clock da memória
    *feedback: A UC atua como o maestro do sistema, orquestrando o fluxo de dados.*

6. Os Registradores são:
    - [ ] Grandes discos para guardar fotos e vídeos
    - [ ] Pequenas baterias para manter a hora do sistema
    - [x] Espaços de memória ultra-rápidos dentro do processador para uso imediato
    - [ ] Cabos de conexão com a impressora
    *feedback: Localizados no topo da hierarquia de memória, são as memórias mais rápidas do computador.*

7. Em relação à persistência dos dados, a Memória RAM é classificada como:
    - [ ] Permanente
    - [ ] Magnética
    - [x] Volátil (perde os dados se faltar energia)
    - [ ] Não-volátil
    *feedback: A RAM serve para guardar dados de trabalho enquanto o computador está ligado.*

8. No Ciclo de Instrução da CPU, qual o passo que ocorre logo após a Busca (Fetch)?
    - [ ] Armazenamento (Store)
    - [ ] Escrita (Write)
    - [x] Decodificação (Decode)
    - [ ] Impressão
    *feedback: O processador primeiro busca os bits na RAM e depois precisa "traduzir" o que eles significam.*

9. Qual barramento é responsável por indicar **AONDE** um dado deve ser buscado ou gravado na RAM?
    - [ ] Barramento de Dados
    - [x] Barramento de Endereço
    - [ ] Barramento de Energia
    - [ ] Barramento Serial
    *feedback: O endereço funciona como o número de uma casa que a CPU quer visitar.*

10. O que define a velocidade de um processador (ex: 3.5 GHz)?
    - [ ] O tamanho físico do chip
    - [ ] A quantidade de núcleos (Cores)
    - [x] O Clock (número de ciclos de instrução realizados por segundo)
    - [ ] O preço de mercado
    *feedback: GHz (GigaHertz) significa bilhões de ciclos de processamento por segundo.*
