# 🧠 Quiz 01: Introdução à Computação e Bases Numéricas

Teste seus conhecimentos iniciais sobre como os computadores processam informações e operam com bits.

1. O que melhor define o conceito de "Computação"?
    - [ ] O ato de digitar rápido em um teclado
    - [ ] A manutenção física de circuitos eletrônicos
    - [x] O processamento automático de informações através de algoritmos
    - [ ] A criação de redes sociais e sites de busca
    *feedback: Computação é a ciência que estuda o tratamento automático da informação por meio de máquinas.*

2. Qual a diferença fundamental entre "Dado" e "Informação"?
    - [ ] Não há diferença, são sinônimos
    - [x] Dados são elementos brutos; Informação é o dado processado e com contexto
    - [ ] Informação é o que entra no PC; Dado é o que sai
    - [ ] Dados são apenas números; Informação é apenas texto
    *feedback: Informação é o resultado do processamento de dados brutos que ganham significado.*

3. Por que o sistema Binário é a "linguagem natural" dos computadores?
    - [ ] Porque é mais fácil de aprender que o decimal
    - [ ] Porque economiza espaço no disco rígido
    - [x] Porque circuitos eletrônicos operam com dois estados (ligado/desligado)
    - [ ] Porque foi inventado antes do sistema decimal
    *feedback: Hardwares usam níveis de tensão (High/Low) que mapeamos para os dígitos 0 e 1.*

4. O que representa o termo "BIT"?
    - [ ] Uma pequena parte de um arquivo
    - [x] Binary Digit (Dígito Binário)
    - [ ] Basic Information Tool
    - [ ] Binary Integration Technology
    *feedback: Um BIT é a menor unidade de informação em computação, assumindo valor 0 ou 1.*

5. Quantos estados diferentes podem ser representados por um conjunto de 8 bits (1 Byte)?
    - [ ] 8 estados
    - [ ] 64 estados
    - [ ] 128 estados
    - [x] 256 estados
    *feedback: Como cada bit tem 2 estados, 8 bits permitem $2^8 = 256$ combinações únicas.*

6. Qual o valor decimal do número binário `1010`?
    - [ ] 8
    - [x] 10
    - [ ] 12
    - [ ] 5
    *feedback: Aplicando pesos ($2^3=8$ e $2^1=2$), temos $8+2=10$.*

7. No sistema Hexadecimal, qual letra representa o valor decimal 15?
    - [ ] A
    - [ ] E
    - [x] F
    - [ ] X
    *feedback: A=10, B=11, C=12, D=13, E=14, F=15.*

8. O que caracteriza um Sistema Numérico Posicional?
    - [ ] Todos os dígitos valem o mesmo independente da ordem
    - [x] O valor de um dígito muda conforme sua posição no número
    - [ ] Usa apenas números positivos
    - [ ] Só pode ser usado na base 10
    *feedback: No sistema posicional, um '1' na casa das centenas vale muito mais que um '1' na casa das unidades.*

9. Qual sistema é amplamente utilizado para representar cores no desenvolvimento Web?
    - [ ] Binário
    - [ ] Octal
    - [x] Hexadecimal
    - [ ] Decimal
    *feedback: Cores RGB são representadas em Hexa (ex: #FFFFFF) por ser compacto e prático.*

10. Se um sistema utiliza a Base 8, quais os dígitos disponíveis?
    - [ ] 0 a 8
    - [x] 0 a 7
    - [ ] 1 a 8
    - [ ] 0 e 1
    *feedback: O sistema Octal usa a base 8, portanto os símbolos vão de 0 até (Base - 1), ou seja, de 0 a 7.*
