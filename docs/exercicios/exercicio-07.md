# 💪 Exercícios — Aula 07: Aritmética Binária

Pense como uma calculadora eletrônica e pratique as operações básicas que ocorrem dentro de toda CPU.

## 🟢 Nível Básico

### 1. Regras da Soma
Complete os resultados das seguintes operações simples de 1 bit:
- a) 0 + 0 = ?
- b) 0 + 1 = ?
- c) 1 + 1 = ?
- d) 1 + 1 + 1 = ?

### 2. Soma Sem Transporte
Realize as somas binárias abaixo. Atenção: estas somas não geram "vai-um" (carry) entre as colunas.
- a) `1010` + `0101`
- b) `1100` + `0011`

---

## 🟡 Nível Intermediário

### 3. O Desafio do Carry (Vai-um)
Realize as somas abaixo levando em conta o "vai-um" para a coluna da esquerda:
- a) `101` (5) + `011` (3)
- b) `111` (7) + `001` (1)
- c) `1011` (11) + `1011` (11)

### 4. Subtração e Empréstimo
Resolva as subtrações binárias usando a técnica do "borrow" (pedir emprestado):
- a) `110` (6) - `010` (2)
- b) `1010` (10) - `0001` (1)

---

## 🔴 Desafio Master

### 5. O Estouro (Overflow)
Imagine um sistema antigo que só consegue armazenar números de **4 bits** (máximo decimal 15).
- a) O que acontece se tentarmos somar `1000` (8) + `1000` (8)?
- b) Qual seria o resultado "visível" para o sistema e como chamamos esse erro técnico?
